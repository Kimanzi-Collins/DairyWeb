## ✨ What is DairyWeb?

**DairyWeb** is a web app designed to make dairy data management easy and engaging.

It is connected to a **local SQL Server database** and provides:
- Beautiful **input forms** for entering daily operational data.
- Professional-looking **reports and dashboards** for quick insights.
- A beginner-friendly experience that feels modern and visually stunning, even for a rookie user.

---

## 🧩 Core Idea

This project is built around a simple but powerful flow:

1. Enter data through clean, guided forms.
2. Save records into a local Microsoft SQL Server instance.
3. Generate clear reports that make decision-making easier.

## 🔗 Database Connection (Local SQL Server)

DairyWeb uses a **local SQL Server** database running on your machine/network.

Typical local setup examples:
- `localhost`
- `localhost\SQLEXPRESS`
- `127.0.0.1`

Make sure your backend connection string targets your local SQL Server instance and the correct database.
